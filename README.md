# 플러터 온라인 스터디

본 프로젝트는 플러터 온라인 스터디를 위한 것입니다.

목표는 간단한 영어 단어장을 만드는 것입니다.

## 프로젝트 참여 방법

1. Git Repo 에 개발자로 등록합니다. 단톡방에 요청해주세요.
2. 개발은 반드시 자신의 'branch' 에서 해 주세요. 'master' 는 가능한 건드리지 마세요.
3. 프로젝트 관리자로 부터 역활을 분담 받으시고 작업을 진행해주세요.
